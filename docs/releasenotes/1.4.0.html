What's New in 1.4

1.4.0

  • System Chrome window mode — a new native macOS window option with real resizing, dragging, and system blur. Now the default for new installs. Switch in Settings > General.
  • Customizable display templates — control what text appears in the header, session rows, and detail lines using tokens like {cwd}, {state}, {model}, and {total_tokens}. Configure in Settings > Appearance.
  • Model and token usage tracking — see which model each session is using and how many tokens it has consumed, read directly from Claude Code transcripts.
  • Native menu bar — the menu bar dropdown now uses standard macOS menu items instead of a custom window.
  • About pane — version info, links, and acknowledgements in Settings.
  • Settings window resizes per tab — each settings pane gets the right amount of space.
  • App is now signed and notarized — no more Gatekeeper warnings on first launch.

Earlier in 1.3

  • Drag snapping — overlay snaps to default position, screen edges, and corners with visual guides
  • Auto-updates — update dialog on launch when a new version is available
  • Collapsed groups remembered — session group state persists across launches

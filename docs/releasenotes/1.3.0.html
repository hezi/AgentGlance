What's New in 1.3

1.3.0

  • Auto-updates via Sparkle — checks for updates automatically with a "Check for Updates" menu item
  • Session grouping — group sessions by project or status with collapsible headers
  • Improved question UX — one-at-a-time mode with prev/next navigation; tap to expand long text
  • Subtle menu bar indicator — gentle pulsing dot instead of spinning arc
  • Move to Applications prompt — one-click move if running outside /Applications
  • Empty state overlay — helpful view with Settings and Setup when no sessions are active
  • Process liveness checks — ghost sessions cleaned up automatically

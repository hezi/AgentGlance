What's New in 2.0

2.0.0

  • Web Remote — monitor sessions and approve permissions from your phone over the local network. Pair with a rotating code, manage devices in Settings > Remote.
  • Markdown completion cards — agent completion messages now render headings, bullet lists, and inline formatting instead of plain text. Expand long messages with Show More.
  • Double-tap to focus terminal — double-tap a session row in the notch overlay to jump straight to its terminal.
  • Smarter notch collapse — the overlay now waits 300ms before collapsing so you can grab resize handles without the window disappearing.
  • Notification opt-out — disable system notifications in Settings > General.
  • Brighter idle icons — idle and unknown status icons in the menu bar are easier to see.
  • Tooltips on notch buttons — pin and settings buttons now show tooltips on hover.
  • Faster terminal activation — Ghostty TTY cache is pre-warmed so switching to a terminal feels instant.

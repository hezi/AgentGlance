AgentGlance

Monitor your AI coding agents from a floating macOS overlay. Approve, answer, review — without breaking flow.

View on GitHub Download latest release → macOS 14+ · Apple Silicon · Zero dependencies

Features

Working Needs Approval Ready

Approve or Deny

Allow, Always Allow, Deny, or Skip tool requests directly from the overlay. No terminal switching.

Answer Questions

When your agent asks a question, pick from selectable options right in the overlay.

Review Plans

See a preview of proposed implementation plans. Approve, reject, or open the full file.

Terminal Navigation

Click any session to jump to the right terminal tab. Works with:

Ghostty Terminal.app iTerm2 Kitty

Clickable URLs

WebFetch URLs and WebSearch queries show as clickable links. See what your agent is browsing.

Multi-Monitor

Pin to a specific screen, follow your cursor, or stick to the main display.

Global Hotkey

Press ⌥⇧C from anywhere to jump to the most urgent session.

Customizable

Font size, expanded width, Liquid Glass mode, dark/light theme, draggable position.

Coming Soon

Support for OpenAI Codex CLI and Google Gemini CLI — same overlay, all your agents in one place.

Get started in 30 seconds

Clone, build, paste the hooks config, and you're running.

git clone https://github.com/hezi/AgentGlance.git
cd AgentGlance
open AgentGlance.xcodeproj
# Cmd+R to build and run, then click "Setup Hooks" in the menu bar
